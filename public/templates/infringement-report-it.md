# Modello Segnalazione Infrazione UE (Italiano)

Usa questo modello per preparare una segnalazione da inviare alla Commissione o per la revisione interna prima dell'invio.

---

Titolo del caso:
Paese: (Italia / Francia / Germania)
Regione / Comune:
Riferimento normativo principale (Direttiva / Articolo):
Sintesi della violazione sospetta (una frase):

Cronologia (date di superamenti, atti, interventi):

Elementi oggettivi (includere link diretti alle fonti):
- Dati ufficiali (ARPA, ISPRA) + URL dataset + ID centralina
- Atti amministrativi (delibere regionali, autorizzazioni) con link
- Provvedimenti giudiziari o procedure aperte (link)
- Fotografie georeferenziate (nomi file)
- Report scientifici o perizie (link)

Prove quantitative richieste (allega CSV/Tabelle):
- Tabelle annuali/giornaliere dei superamenti (fonte + anni)
- Stato di funzionamento impianti (log attivo/inattivo)
- Registri di scarico (volumi / concentrazioni)

Checklist prima dell'invio:
- [ ] Link alle fonti principali archiviati (data download)
- [ ] File dati (CSV/XLS) allegati e descritti
- [ ] Foto georeferenziate e datate
- [ ] Atti amministrativi identificati (autorità, data, link)
- [ ] Contatti locali indicati
- [ ] Dati personali rimossi o oscurati per sicurezza

Passi suggeriti (possiamo assistere):
1. Prepara una lettera di accompagnamento sintetica.
2. Allegare il pacchetto di prove (CSV, foto, copie atti).
3. Apri una issue su GitHub o invia via email (indirizzo nel file README).
4. In presenza di danno immediato, contatta autorità ambientale locale e ufficio stampa.

Note su sicurezza & verifica:
- Prediligi fonti ufficiali (ARPA, ISPRA, portali regionali, Gazzetta Ufficiale).
- Archivia snapshot (Wayback o PDF) delle pagine che possono cambiare.
- Se sei un whistleblower, non inserire materiale identificativo negli allegati: contattaci prima per canali sicuri.
