# Modèle de signalement d'infraction UE (Français)

Utilisez ce modèle pour préparer un signalement adapté au dépôt d'une plainte administrative ou pour une revue interne avant soumission à la Commission.

---

Titre :
Pays : (Italie / France / Allemagne)
Région / Commune :
Référence juridique principale (Directive / Article) :
Résumé de la violation suspectée (une phrase) :

Chronologie (dates des événements, dépassements, mesures officielles) :

Preuves objectives (liste avec sources directes) :
- Jeu de données de surveillance officiel (nom de l'agence + URL du dataset + identifiant de station)
- Actes administratifs (liens vers arrêtés régionaux / permis / autorisations)
- Décisions judiciaires ou procédures existantes (liens)
- Photographies / preuves géolocalisées (noms de fichiers)
- Rapports scientifiques, vérifications tierces (liens)

Preuves quantitatives requises (joindre tableaux / CSV) :
- Tableau des dépassements annuels/journaliers (source + années)
- État de fonctionnement des installations (logs actives/inactives)
- Registres de rejets (volume / concentrations polluantes)

Checklist avant soumission :
- [ ] Liens vers sources primaires inclus et archivés (date de téléchargement)
- [ ] Fichiers de données (CSV/XLS) joints et décrits
- [ ] Photos géoréférencées et horodatées
- [ ] Actes administratifs identifiés (autorité, date, lien)
- [ ] Contacts locaux/régionaux listés
- [ ] Données personnelles supprimées ou expurgées pour sécurité

Étapes suggérées (nous aidons) :
1. Préparer une courte lettre d'accompagnement décrivant la violation.
2. Joindre le dossier de preuves (CSV, photos, copies des actes).
3. Ouvrir une issue sur GitHub dans le dépôt ou nous envoyer un email avec le dossier.
4. Si un préjudice immédiat est constaté, avertir l'autorité environnementale locale et le service de presse.

Notes sur sécurité et vérification :
- Utilisez des sources officielles (ARPA, ISPRA, portails régionaux, Journal officiel).
- Archivez des captures (Wayback ou PDF) des pages susceptibles de changer.
- Si vous êtes lanceur d'alerte, n'incluez pas d'informations identifiantes dans les fichiers publics — contactez-nous d'abord pour une prise en charge sécurisée.

## Checklist rapide des preuves par déclencheur

- Retards de paiement publics (Directive 2011/7/UE):
	- [ ] Export des factures (date émission / date paiement) avec source et timestamp
	- [ ] Tableau agrégé (CSV) montrant % paiements > 60 jours
	- [ ] Rapport de la Cour des comptes mentionnant des retards

- Attributions directes & fragmentation (Directives 2014/24/UE, 2014/25/UE):
	- [ ] Tableau des attributions répétées (même fournisseur/CIG)
	- [ ] Copies des décisions d'attribution et avis
	- [ ] Preuves de fragmentation artificielle (descriptions similaires, dates proches)

- VIA/VAS (Directives 2011/92/UE, 2001/42/CE):
	- [ ] Chronologie des délais statutaires vs dates réelles de décision
	- [ ] Dossier projet et absence d'acte de suspension
	- [ ] Liens vers portails VIA régionaux
